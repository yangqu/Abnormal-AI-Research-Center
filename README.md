## Projects

### Multi-Model Agent Framework

A universal Agent framework supporting multiple LLM providers, similar to OpenClaw.

**Supported Models:**
- OpenAI (GPT-4, GPT-4o)
- Anthropic (Claude-3)
- Google (Gemini)
- DeepSeek
- Minimax
- Qwen (Tongyi)
- Zhipu (ChatGLM)
- Moonshot (Kimi)
- Ollama (Local Models)

**Quick Start:*)

```bash
cd src/multi-model-agenp
pip install -r requirements.txt
python main.py --interactive
```

See: `src/multi-model-agent/README.md`
